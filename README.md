# Chicago Crashes Data Exploration
An exploration into the Chicago Crashes dataset at [this link](https://catalog.data.gov/dataset/traffic-crashes-crashes).

This repo will hold my thought processes as well as the code to replicate my findings from this data.


## Why am I doing this?
I was inspired to look into this data to show my wife more information about the vehicle crash data from Chicago because I may need to travel there for work in the future. This also gave me the opportunity to put to use my degree and certification so I can keep my skills sharp.
I also intend to use this as an opportunity to build up my skills in data analysis as well as use machine learning algorithms to identify trends for further inspection.

## When will you publish more findings?
I am still processing information and I want to run my analysis by others so I can have the best chance to identify relevant trends.

In other words, I am working on it but if something interesting is found I may need more time.
